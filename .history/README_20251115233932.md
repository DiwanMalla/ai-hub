### ğŸ§© Adding More Models

The platform is designed to be extensible. To add more AI models:

1. Browse [Hugging Face Models](https://huggingface.co/models) compatible with Transformers.js
2. Update the `aiModels` array in `src/app/page.tsx`
3. Create a new component in `src/components/` for the model's UI
4. Import and use the model pipeline from `@huggingface/transformers`

**Supported tasks include:**
- Image segmentation
- Object detection
- Image classification
- Text generation
- Summarization
- Translation
- And many more!
